AI200 Capstone Project

In this project, we applied Machine Learning skills on Loan Default Prediction Competition using Dataset from Kaggle.
We have done the followings for this project which classified as Classification Problem:
1. Import all the libraries.
2. Import dataset from csv to pandas df.
3. Check the data of each columns if there is any data that we need to clean/update.
4. Data Cleaning.
5. Exploratory Data Analysis (EDA) to understand which features/columns/variables are important, view summary statistics and visualize the data.
6. Feature Engineering. We select relevant features and enhance them to improve the overall performance of the machine learning model.
7. Model building and comparing model performance. We use Train Test Split and KFold Cross Validation on the following models:
- Logistic Regression Model.
- K-Nearest Neighbors (KNN) Model.
- Decision Tree Model.
- Random Forest Model.
- Gradient Boosting Model.
- XGBoost Model.
- CatBoost Model.
8. We choose the final model then we generate and export predictions from final model and submit it to kaggle.

AI300 Capstone Project
